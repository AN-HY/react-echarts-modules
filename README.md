### react15.6.1实现echarts模块化加载

**我们真的需要react-echarts插件吗？**

## NO

**在这里，我使用echarts提供的模块化加载方式，实现了几个react-echarts图表组件**

#### 插件版本号

```json
  "echarts": "^3.6.2",
  "react": "^15.6.1",
  "react-dom": "^15.6.1"
```

#### 实现了哪些图表组件

1、饼图

2、柱状图

3、折线图

4、散点图

5、地图

6、雷达图

7、k线图
